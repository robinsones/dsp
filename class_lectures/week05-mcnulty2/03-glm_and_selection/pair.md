# Merging sorted lists

you are given two sorted lists, of length m and n. Write a function that will merge the two lists into a single sorted list.

Input
    [1,3,5,6]
    [2,4,7]
Output
    [1,2,3,4,5,6,7]

Input
    [15]
    [12]
Output
    [12,15]


Input
    []
    [1,2]
Output
    [1,2]
